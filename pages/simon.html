The Simon Game

A course project

I built a browser‑based Simon Game as part of my Udemy course to sharpen my understanding of JavaScript logic, event handling, and sequencing. The project challenged me to track game state, generate and store color patterns, and compare user input step‑by‑step against the correct sequence. It also pushed me to work with sound effects, animations, and timing to recreate the classic feel of the original game. Completing it gave me a much better grasp of how to structure interactive features and manage more complex user interactions in the browser.
